# Frontend Responsive (Mobile-First)

Documentación oficial del rediseño responsive del frontend para uso operativo en celulares.

## Objetivo

Pasar de una experiencia mayormente desktop a una experiencia **mobile-first** que permita operar el negocio desde el teléfono sin perder funcionalidades clave: pedidos, caja, productos, clientes, gastos e indicadores.

## Documentos de esta carpeta

- [USABILIDAD_PRODUCTO.md](./USABILIDAD_PRODUCTO.md)
  - Guía funcional para dueños/encargados y equipo operativo.
  - Explica cómo usar la app en celular y cuáles son los flujos recomendados.

- [ARQUITECTURA_TECNICA.md](./ARQUITECTURA_TECNICA.md)
  - Guía técnica para desarrolladores y agentes de IA.
  - Describe decisiones de arquitectura responsive, componentes modificados, convenciones y checklist de evolución.

## Alcance implementado

Incluye dos etapas de mejoras:

1. Base responsive global (layout, navegación mobile, tablas/card views, login, páginas principales).
2. Segunda pasada UX en flujos críticos (crear/editar pedido, caja, formularios complejos).
